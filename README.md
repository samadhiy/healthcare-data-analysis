📊 Healthcare Data Analysis – Big Data Project

📌 Overview

This project was developed as part of the CCS3309 Big Data module. It focuses on analyzing healthcare data to extract meaningful insights using Exploratory Data Analysis (EDA) and visualization techniques.

The project demonstrates how data can be transformed into valuable information to support better understanding and decision-making.

🎯 Objectives
* Perform Exploratory Data Analysis (EDA)
* Identify trends and patterns in healthcare data
* Apply data visualization techniques to interpret results


🛠️ Technologies Used

🐍 Python

📓 Jupyter Notebook

📊 Pandas

🔢 NumPy

📈 Matplotlib

🎨 Seaborn
